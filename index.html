MathPDF - 免费在线Markdown转PDF工具

MathPDF 是一款专业的在线 Markdown 转 PDF 工具，完美支持 LaTeX 数学公式渲染。无需注册，无水印，浏览器端直接转换。

主要功能

  • 完美渲染 LaTeX 数学公式，支持 amsmath、amssymb、mhchem 等包
  • 支持中英文双语内容，使用思源宋体等专业中文字体
  • 代码高亮显示，支持多种编程语言
  • 多种预览主题：默认、学术、简约、优雅
  • 支持 A4、Letter、Legal 等多种纸张尺寸
  • 可调节页边距：正常、窄边距、宽边距
  • 支持导出为 PDF 或独立 HTML 文件
  • 支持图片粘贴和拖拽嵌入
  • 完全免费，无需注册，无使用限制
  • 浏览器端处理，保护隐私安全

适用场景

MathPDF 适用于学术写作、技术文档、数学作业、科研论文、教学材料等需要包含数学公式的文档转换场景。

如何使用

  1. 打开 MathPDF 网站
  2. 在左侧编辑器中输入或粘贴 Markdown 内容
  3. 使用 $行内公式$ 或 $$显示公式$$ 语法添加数学公式
  4. 在右侧预览效果
  5. 点击"导出 PDF"按钮或按 Ctrl+P 生成 PDF 文件

支持的数学公式

支持行内公式如 $E = mc^2$ 和显示公式如积分、求和、矩阵、化学方程式等。支持 \frac{}{}、\sqrt{}、\sum、\int、\lim 等常用数学符号。

常见问题

MathPDF 是免费的吗？

是的，MathPDF 完全免费使用，无需注册，无水印，无使用次数限制。

支持哪些 LaTeX 包？

支持 amsmath、amssymb、mhchem 等常用数学和化学公式包。

可以导出为其他格式吗？

除了 PDF，还支持导出为独立的 HTML 文件，包含完整的样式和数学公式渲染。

支持中文吗？

完全支持中文内容，使用 Source Han Serif SC 和 Noto Serif CJK SC 等专业中文字体，确保中文排版美观。
